# Blinkit-Power-BI
📊Blinkit Sales and Operations Analysis Dashboard (Power BI)

This project focuses on analyzing Blinkit's sales and operational data using Power BI to derive meaningful business insights and support data-driven decision-making.

🔍 Overview

Developed an interactive Power BI dashboard to analyze key performance metrics such as sales trends, product performance, outlet types, and customer preferences. The goal was to transform raw data into actionable insights for business growth.

🎯 Objective
To build a business intelligence solution that helps stakeholders quickly understand performance metrics and make informed decisions.

🚀 Key Features
Sales performance analysis across different outlet types and locations
Product category-wise insights and top-performing items
Visualization of revenue trends and customer buying patterns
KPI tracking (Total Sales, Average Sales, Number of Items, Ratings)
Interactive filters and slicers for dynamic data exploration

🛠 Tools & Technologies
Power BI
Excel / CSV Dataset
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)

📈 Insights Derived
Identified high-performing outlet types and locations
Analyzed customer preferences across product categories
Discovered trends in sales and item visibility
Helped in understanding factors affecting product ratings and sales
